## 1枚の画像が、寝ている間に売れ続ける。

ストックフォト販売の魅力は「不労所得」です。

1度アップロードした画像が、何度もダウンロードされるたびにお金が入る。
・
・
・
100枚アップロードして、月に30回ダウンロード。**月3,000〜15,000円の不労所得**。

「少なくない？」と思いましたか？

ここからがポイントです。AIを使えば、100枚の画像を**1日で生成**できる。従来のカメラマンが100枚撮影するのに数週間かかることを考えると、圧倒的に効率がいい。

500枚、1,000枚とストックを積み上げていけば、月3〜5万円のストック収入は現実的な数字です。

しかも一度アップすれば、あとは何もしなくてもダウンロードされるたびにチャリンチャリンと入ってくる。

### ストックフォト販売の全体フロー

```
ステップ1：販売プラットフォームに登録
    ↓
ステップ2：売れるジャンルをリサーチ
    ↓
ステップ3：AIで画像を大量生成
    ↓
ステップ4：メタデータを設定してアップロード
    ↓
ステップ5：データを分析して追加生成
```

## 使えるプラットフォームと収益目安

| プラットフォーム | 1DLあたりの収益 | 審査 | AI画像の可否 |
|---|---|---|---|
| **Adobe Stock** | 30〜200円 | あり | AI生成を明示すればOK |
| **PIXTA** | 30〜180円 | あり | AI生成タグ必須でOK |
| **写真AC** | 3〜10円/DL | なし | OK（参入しやすい） |

> **注意**: Shutterstockは外部ツール（Midjourney等）で生成したAI画像の投稿を受け付けていません。自社AI生成ツール（Shutterstock Generate）経由のみ対応です。

2026年3月時点で、主要プラットフォームはAI生成画像の販売を認めています。ただし**必ず「AI生成」であることを明示する必要がある**。これを隠すとアカウントBANのリスクがあります。

おすすめの始め方は**Adobe Stock＋PIXTA**の2つに同時出品。同じ画像を複数プラットフォームに出品するのはOKなので、露出を最大化できます。

## 売れる画像のジャンル

AIストックフォトで売れるのは「リアルな写真」ではなく**「イラスト・コンセプト画像」**です。

### 需要が高いジャンル

**1. ビジネス系イラスト**
- 会議、プレゼン、チームワークのイメージ
- 企業のWebサイトや資料で大量に使われる

**2. テクノロジー系**
- AI、DX、クラウドコンピューティングのコンセプト画像
- IT企業のブログやホワイトペーパーで需要が高い

**3. ライフスタイル系**
- 在宅ワーク、健康、旅行のイメージ
- メディア記事のアイキャッチとして需要が安定

**4. 季節・イベント系**
- 花見、夏祭り、クリスマス等の季節素材
- 時期になると検索が急増する。先回りしてアップしておくのがコツ

「写真っぽい画像は売れないの？」

売れますが、審査が厳しい。AI生成の人物写真は肖像権の問題で審査落ちしやすいです。最初はイラスト系から始めるのが安全。

## AIで画像を効率的に生成する

### 使うツール

| ツール | 強み | 月額料金 |
|---|---|---|
| **Midjourney** | 高品質、商用利用OK | $10〜 |
| **Leonardo.ai** | 無料枠あり、バリエーション豊富 | 無料〜$12 |
| **DALL-E 3** | ChatGPTから使える手軽さ | ChatGPT Plus内 |

Midjourneyが品質面で一歩リードしていますが、月10ドルのコストがかかります。初期費用を抑えたいなら、Leonardo.aiの無料枠から始めてもOK。

### 1日100枚生成のワークフロー

```
効率的な画像生成フロー：

  1. 売れ筋キーワードを10個リストアップ（10分）
  2. 各キーワードでプロンプトを3パターン作成（15分）
  3. AIで画像生成：30プロンプト×3〜4バリエーション = 100枚（30分）
  4. 品質チェック＋選別（15分）
  5. メタデータ（タイトル・タグ）設定（30分）

  合計：約1時間40分で100枚
```

ポイントはメタデータの設定。タイトルとタグを適切につけないと、検索で表示されません。ここにもAIを活用します。画像のテーマをChatGPTに伝えて「ストックフォト用のタイトルと英語タグを20個生成して」と依頼すればOK。

## 審査に通すコツ

各プラットフォームの審査で落ちやすいポイントと対策です。

**落ちる原因1：品質が低い**
→ 対策：解像度を最大設定で生成。ノイズやアーティファクトがある画像は除外。

**落ちる原因2：AI生成の申告漏れ**
→ 対策：アップロード時に必ず「AI生成」にチェック。これは必須です。

**落ちる原因3：類似画像が多すぎる**
→ 対策：同じプロンプトのバリエーションは3枚まで。構図やカラーを変えて差別化。

**落ちる原因4：商標・ブランドが写り込んでいる**
→ 対策：プロンプトにブランド名を入れない。生成後も確認。

LINEスタンプ販売と並行して取り組むと、AI画像スキルが相乗効果で伸びます。[AIでLINEスタンプを作って販売する方法](/blog/ai-line-stamp-guide)も参考にしてください。

## 収益を最大化する戦略

最初の100枚をアップしたら、データ分析フェーズに入ります。

```
データ分析→改善のサイクル：

  月1回、ダウンロード数を確認
      ↓
  売れている画像のジャンル・スタイルを特定
      ↓
  そのジャンルで追加の画像を50〜100枚生成
      ↓
  売れないジャンルは撤退
      ↓
  繰り返し → 3ヶ月で500枚 → 月3〜5万円のストック収入
```

「3ヶ月で月3〜5万円って遅くない？」

ストックフォトは即金ではなく**積み上げ型**の収益です。半年後、1年後に効いてくる。他の副業と並行して取り組むのがベストです。[月5万円ロードマップ](/blog/ai-side-job-roadmap-monthly-50k)で紹介した「3本柱」の1本として組み込んでください。
