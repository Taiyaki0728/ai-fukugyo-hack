## 時給5,000円の副業。しかも「教えるだけ」。

「AI副業って結局、作業系の仕事でしょ？」

違います。2026年、最も単価が高いAI副業は**「教える」仕事**です。

具体的には、中小企業に対して「ChatGPTの使い方」を教えるコンサルティング。
・
・
・
1回2時間のオンラインセッションで**1万円**。月に8回やれば8万円。さらに月額顧問契約を1社取れば、それだけで月12万円。合計**月20万円**。

「自分にコンサルなんてできるわけない」と思いましたか？

ここが面白いところです。AI活用コンサルに必要なのは「AIの専門知識」ではなく、**「普通にAIを使った経験」**。このサイトを読んでAI副業を3ヶ月やった人は、中小企業の社長より遥かにAIに詳しい。

## なぜ今「AI活用コンサル」が狙い目なのか

2026年3月時点で、日本の中小企業のAI導入率は**まだ20%以下**。

つまり80%の企業が「AIを使いたいけど、やり方がわからない」状態。しかも大手コンサルファームに頼むと月50万〜100万円。中小企業には手が出ない。

ここに「月5万〜10万円で実務レベルのAI活用を教えてくれる個人コンサルタント」の需要があります。

「でも企業相手って難しそう」と思いますよね？

実は中小企業の社長が求めているのはこんなレベルのサポートです。

- ChatGPTで議事録を要約する方法
- AIで営業メールのテンプレートを作る方法
- Canva AIで社内資料のデザインを改善する方法

どれも、AI副業を3ヶ月やった人なら余裕で教えられる内容です。

## コンサルメニューの設計

最初から複雑なメニューを作る必要はありません。以下の3つだけでスタートできます。

```
コンサルメニュー構成：

  ① 単発セッション（2時間）      → 1万円
     内容：ChatGPTの基本操作＋業務での活用法3つ

  ② 3回パック（2時間×3回）      → 2.5万円
     内容：基本操作＋業務フロー改善＋社内定着サポート

  ③ 月額顧問契約                → 5〜10万円/月
     内容：月2回のセッション＋チャットでの質問対応
```

おすすめの流れは「単発→3回パック→月額顧問」のアップセル。単発で信頼を得てから、継続契約に移行するのが最もスムーズです。

## 最初のクライアントを獲得する方法

「どうやって企業クライアントを見つけるの？」

いきなりテレアポする必要はありません。最初の1〜3社は以下の方法で獲得できます。

**方法1：知人・前職の繋がり**

意外とこれが一番早い。前職の同僚や友人の会社に「AI活用の勉強会、無料でやりますよ」と提案する。無料勉強会が良ければ、有料のコンサル契約に移行できます。

**方法2：ココナラ・ストアカで出品**

「ChatGPTの使い方を教えます」というサービスを出品。最初は1回3,000円でもOK。実績とレビューを5件集めたら単価を上げる。[月5万円ロードマップ](/blog/ai-side-job-roadmap-monthly-50k)で解説した「最初は実績優先」の考え方はコンサルでも同じです。

**方法3：SNSで発信→問い合わせ獲得**

Threads、X、LinkedInで「AIの業務活用Tips」を毎日発信。3ヶ月続ければ問い合わせが来始めます。SNS自動運用の方法は[AIエージェントで副業を自動化する方法](/blog/ai-agent-side-job)を参考に。

## セッションの進め方

初めてのコンサルセッションで何を話せばいいのか。テンプレートを用意しました。

```
2時間セッションの構成：

  0:00〜0:15  ヒアリング（現状の業務課題を聞く）
  0:15〜0:30  デモ（ChatGPTの基本操作を見せる）
  0:30〜1:00  ハンズオン（参加者が実際に触る）
  1:00〜1:30  業務適用（「御社のこの業務に使えます」を3つ提案）
  1:30〜1:50  Q&A
  1:50〜2:00  次回提案（3回パックor月額顧問の案内）
```

ポイントは「教える」のではなく「一緒にやる」こと。画面共有しながらリアルタイムでChatGPTを使って、参加者の実際の業務課題を解決してみせる。これが一番刺さります。

## 月20万円を安定させる

```
月20万円の内訳（例）：

  月額顧問A社  ██████████████  10万円
  月額顧問B社  ████████        5万円
  単発セッション ██████          3万円（3回分）
  教材販売     ████            2万円（note等）
  ──────────────────────────────────
  合計                        20万円
```

月額顧問を2社確保すれば、15万円はほぼ固定収入。残り5万円を単発セッションとコンテンツ販売で埋める。

教材販売には「AI活用マニュアル」をnoteで売る方法が相性いいです。詳しくは[AIプロンプトを販売して月3万円稼ぐ方法](/blog/ai-prompt-selling)を参照。

「でもコンサルって副業でできるの？時間的に」

月額顧問2社で月4回のセッション（各2時間）＋チャット対応で月16〜20時間。十分副業の範囲内です。

確定申告については[AI副業の確定申告完全ガイド](/blog/ai-side-job-tax-guide)を必ず読んでおいてください。月20万円なら開業届を出したほうが節税できるケースが多いです。
