## 「AIへの指示文」が売れる時代が来た。

プロンプト。つまりAIへの指示文。

これ、お金になります。
・
・
・
noteで「ChatGPT用プロンプト集」を1,980円で販売。月15部売れて**月29,700円**。

「指示文を書くだけで3万円？」

そうです。ただし「適当なプロンプトを並べたもの」は売れません。売れるのは**「特定の悩みを解決するプロンプト」**です。

例えば「営業メールを10パターン自動生成するプロンプト」「ブログ記事の構成案を3分で作るプロンプト」「Instagram投稿文を1週間分まとめて作るプロンプト」。

こういう「コピペして使えば即結果が出る」プロンプトに、人はお金を払います。

この記事では、プロンプトの商品化から販売、集客までの全フローを解説します。

---

## 売れるプロンプトの3条件

まず「どんなプロンプトが売れるのか」を理解してください。

```
売れるプロンプトの条件：

  ① 特定の悩みを解決する（汎用的すぎるのはNG）
  ② コピペで使える（設定や前提知識が不要）
  ③ 結果の質が明らかに高い（普通に聞くより良い出力）
```

**売れる例：**
- 「転職面接の想定質問と回答を自動生成するプロンプト」
- 「Kindle本の目次構成を5分で作るプロンプト」
- 「SEO記事の構成案を競合分析付きで作るプロンプト」

**売れない例：**
- 「ChatGPTを便利にするプロンプト100選」（広すぎて価値が薄い）
- 「面白い回答をさせるプロンプト」（娯楽系は単価が取れない）

ポイントは**「ビジネスの具体的な課題を解決する」**こと。お金を払うのはビジネスユースの人です。

## プロンプト商品の設計方法

プロンプト1本で売るのではなく、**「プロンプト集＋使い方解説」のパッケージ**にします。

### 商品の構成

```
プロンプト商品の構成（例：ブログライター向け）：

  1. 商品タイトル：「AI×ブログ記事作成プロンプト完全パック」
  2. 内容：
     ├── プロンプト①：キーワードリサーチ用
     ├── プロンプト②：記事構成案作成用
     ├── プロンプト③：本文執筆用
     ├── プロンプト④：リライト・校正用
     └── プロンプト⑤：メタディスクリプション生成用
  3. 各プロンプトの使い方解説（スクリーンショット付き）
  4. カスタマイズ方法の説明
  5. 実際の出力サンプル
```

プロンプト5本＋解説で1,980〜2,980円が相場。これが月10〜20部売れるイメージです。

「5本だけで2,000円？高くない？」

逆です。5本でも「それを使えば1時間の作業が10分になる」なら、むしろ安い。価値は「プロンプトの数」ではなく「解決する問題の大きさ」で決まります。

## noteでの販売手順

販売プラットフォームはnoteが最適です。理由は3つ。

1. 登録・出品が無料
2. 手数料は合計約15〜25%（クレカ決済なら約14.5%）
3. note内での検索流入が見込める

### 出品の手順

**1. note記事の前半を無料公開する**

記事の最初30〜40%は無料で読めるようにする。ここで「このプロンプトを使うとこんな結果が出る」という実例を見せる。価値を先に証明するのが重要。

**2. 有料部分にプロンプト本体を設置**

実際のプロンプトと使い方解説は有料エリアに。価格は最初1,480〜1,980円がおすすめ。

**3. 購入者特典をつける**

「アップデート時は無料で差し替え版を提供」「質問はDMで対応」など。特典があると購入率が上がる。

## 集客：noteの外から人を連れてくる

note内の検索だけでは売上が安定しません。外部からの流入を作ります。

**SNSで「プロンプトの一部」を無料公開する**

Threads、X、Instagramで「このプロンプトを使ったらこんな結果が出た」という投稿をする。プロンプトの一部だけ見せて、全文はnoteへ誘導。

SNS運用の自動化については[Threads×AI完全自動運用で月3万円稼いだ方法](/blog/ai-thread-automation)が参考になります。

**ブログからの導線**

もしブログを運営しているなら、関連記事からnoteの販売ページへ誘導する。ブログアフィリエイトと組み合わせる方法は[ChatGPTでブログアフィリエイト完全ガイド](/blog/chatgpt-blog-affiliate-guide)で解説しています。

## 月3万円の収益モデル

```
月3万円の内訳：

  プロンプト集A（ライター向け） 1,980円×8部 = 15,840円
  プロンプト集B（SNS運用向け） 1,480円×6部 = 8,880円
  単品プロンプト（各種）       500円×12部  = 6,000円
  ─────────────────────────────────────
  合計                                    = 30,720円
  note手数料（約15%）差引後                 = 約26,100円
```

手数料を引くと実質2.6万円。3万円にするなら、もう1〜2商品追加するか、SNSからの流入を増やして部数を伸ばすかです。

ただしプロンプト販売は**積み上げ型**。商品を増やすほど月の売上が安定します。半年後に商品が10個になれば、月5〜10万円も見えてきます。

他の収入源との組み合わせ方は[月5万円ロードマップ](/blog/ai-side-job-roadmap-monthly-50k)を参考にしてください。
