## 20万円ルールの正体

「副業所得20万円以下は確定申告不要」というルール。これ、正確に理解している人は少ないです。

### 正確なルール

- 給与所得者（会社員）で、副業の**所得**（収入-経費）が年間20万円以下なら、**所得税の確定申告**は不要
- ただし**住民税の申告は必要**（市区町村の役所で手続き）
- 副業所得が20万円を超えたら、所得税の確定申告が必要

ここでの「所得」は「収入」ではないことに注意してください。

```
所得の計算：

  副業の収入          36万円（月3万円×12ヶ月）
  - 経費              12万円（AIツール代、PC按分費、通信費等）
  ──────────────────
  副業所得            24万円 → 20万円超えなので確定申告が必要

  もし経費が17万円なら：
  副業所得            19万円 → 20万円以下なので所得税の確定申告不要
                              （でも住民税の申告は必要）
```

つまり経費をきちんと計上すれば、確定申告が不要になるケースもあります。

## AI副業で経費にできるもの一覧

AI副業に関連する支出は、経費として計上できます。

### 確実に経費になるもの

| 項目 | 具体例 | 目安金額（年間） |
|---|---|---|
| AIツール月額費用 | ChatGPT Plus、Claude Pro、Midjourney | 2.4〜12万円 |
| クラウドソーシング手数料 | クラウドワークス、ランサーズの手数料 | 収入の5〜20% |
| 通信費（按分） | インターネット回線、スマホ代 | 実費の30〜50% |
| PC・周辺機器（按分） | パソコン、モニター、キーボード | 実費の30〜70% |
| 書籍・教材費 | AI関連の書籍、有料教材 | 実費全額 |
| 文房具・消耗品 | ノート、ペン等 | 実費全額 |

### 按分計算のポイント

「按分」とは、プライベートと仕事で兼用しているものの費用を、仕事で使う割合だけ経費にすること。

- PC：副業での使用が50%なら、PC代の50%が経費
- 通信費：副業での使用が30%なら、通信費の30%が経費
- 家賃（在宅作業の場合）：作業スペースの面積割合で按分

「この割合って自分で決めていいの？」

合理的な根拠があればOKです。ただし「PC代の100%を経費にした」は税務調査で否認されるリスクが高い。30〜50%が安全圏です。

## 開業届を出すべきか

副業が軌道に乗ってきたら、個人事業主として開業届を出すかどうかの判断が必要です。

### 開業届を出すメリット

**1. 青色申告特別控除（最大65万円）**

これが最大のメリット。年間所得から最大65万円を差し引ける。所得税率が20%なら、年間13万円の節税になります。

**2. 赤字の繰越し（3年間）**

初年度が赤字（経費が収入を上回った場合）なら、翌年以降の利益と相殺できる。

**3. 経費の幅が広がる**

家賃按分や接待交際費など、開業届なしでは認められにくい経費も計上しやすくなる。

### 開業届を出す目安

```
開業届の判断基準：

  副業所得が年間48万円以下  → 出さなくてOK
  副業所得が年間48〜100万円 → 出したほうが節税メリットあり
  副業所得が年間100万円以上 → 出すべき（青色申告の恩恵が大きい）
```

月5万円の副業なら年間60万円。開業届を出して青色申告すれば、それだけで数万円の節税になります。

## 住民税の申告（最も忘れがち）

所得税の確定申告をすれば、住民税は自動で計算されます。

問題は「所得20万円以下で確定申告をしなかった場合」。この場合、住民税の申告を**自分で市区町村の役所に提出する必要がある**。

### 会社にバレたくない場合

住民税の納付方法を「普通徴収（自分で納付）」に変更すればOK。確定申告書の「住民税に関する事項」で「自分で納付」にチェックを入れるだけです。

ただし、自治体によっては対応が異なるケースもあるので、心配な場合は市区町村の税務課に電話で確認してください。

## 確定申告の具体的な手順

「で、具体的にどうやるの？」

2026年の確定申告は**e-Tax（電子申告）**が最も簡単です。

```
確定申告の手順：

  1. 収入と経費の記録を集計する（1月〜12月分）
  2. 国税庁の確定申告書等作成コーナーにアクセス
  3. 収入金額・経費を入力
  4. マイナンバーカードでe-Tax送信
  5. 完了（還付がある場合は1〜2ヶ月で振り込まれる）
```

帳簿の管理には会計ソフト（freee、マネーフォワードなど）を使うと楽です。月々の経費をこまめに記録しておけば、確定申告時に慌てることはありません。

「AI副業で使ったツールの領収書、全部取っておいたほうがいい？」

はい。サブスクの決済メール、Amazonの購入履歴など、デジタルの記録でOKです。紙である必要はありません。ただし**7年間の保存義務**があるので、フォルダを作って整理しておいてください。
