## YouTube台本1本で5,000〜15,000円。しかも顔出し不要。

「YouTubeで稼ぎたいけど、顔出しも編集もできない」

実は、YouTubeには**顔を出さずに稼げるポジション**があります。

それが「台本ライター」です。
・
・
・
台本1本の相場は**5,000〜15,000円**。月に10本納品すれば月10万円。

しかもChatGPTを使えば、1本の台本を**1〜2時間**で仕上げられる。

「台本って、何を書けばいいの？」

YouTubeの台本は「ブログ記事を話し言葉にしたもの」に近い。ブログ記事が書ける人なら、台本も書けます。AI副業でライティングをやっている人には特に相性がいい。


## YouTube台本の基本構造

YouTube台本には「型」があります。この型を知っているかどうかで、台本の品質が全然違います。

### 10分動画の台本構成

```
YouTube台本の基本構造（10分動画 = 約4,000字）：

  ① 冒頭フック（30秒）     → 視聴者の関心を掴む
     「○○で悩んでいませんか？この動画を最後まで見れば解決します」

  ② 結論の先出し（30秒）   → 動画全体の結論を最初に述べる
     離脱率を下げる最も効果的なテクニック

  ③ 本編（7〜8分）         → 3〜5つのポイントに分けて解説
     各ポイント：主張→理由→具体例→まとめ

  ④ まとめ（1分）          → 3行で要点を振り返る

  ⑤ CTA（30秒）           → チャンネル登録・高評価のお願い
```

この構造をChatGPTに覚えさせておけば、毎回ゼロから指示を出す必要がなくなります。

## ChatGPTで台本を作る具体的フロー

1本の台本（約4,000字）を1〜2時間で仕上げるフローです。

```
台本作成フロー：

  1. テーマのリサーチ（15分）
     → 競合動画の構成をChatGPTで分析

  2. 構成案の作成（10分）
     → 見出し5つ＋各見出しの要点をAIが生成

  3. 本文の下書き（20分）
     → 構成案に基づいてAIが4,000字の台本を生成

  4. 話し言葉への変換（15分）
     → 「です・ます」→「〜なんですよね」等のトーン調整

  5. 人間による編集（30分）
     → 事実確認＋クライアントの動画トーンに合わせた修正

  合計：約1.5時間
```

**最も差がつくのはステップ4と5**。AIの出力は「書き言葉」になりがちです。YouTube台本は「話し言葉」でないと視聴者に刺さらない。ここを人間が調整するのが台本ライターの価値です。

ChatGPTとClaudeの使い分けについて。台本作成ではChatGPTの方が「カジュアルな話し言葉」を生成しやすい印象です。ただし長文の一貫性はClaudeが優秀。詳しくは[ChatGPT vs Claude vs Gemini徹底比較](/blog/chatgpt-vs-claude-vs-gemini-2026)を参照。

## 案件を獲得する方法

YouTube台本の案件は、以下の3つのルートで獲得できます。

**ルート1：クラウドソーシング**

クラウドワークス、ランサーズで「YouTube 台本」と検索。2026年時点で常時100件以上の募集があります。最初の3件は単価より実績優先で。案件獲得のコツは[AI×Webライターで月10万円稼ぐ方法](/blog/ai-writing-crowdsourcing)と共通です。

**ルート2：YouTuberに直接営業**

登録者1,000〜10,000人規模のYouTuberが狙い目。このレベルのYouTuberは「台本を外注したいけど、大手制作会社に頼む予算はない」状態。DMで「台本作成お手伝いできます」と提案してみてください。

**ルート3：SNSでポートフォリオ公開**

Threads、Xで「YouTube台本のサンプル（一部）」を公開する。実際の台本を載せるのはNGですが、構成案やテクニックの解説は自由にできます。

## 単価を上げるステップ

```
単価のステップアップ：

  初心者（実績0〜5件）   → 1本3,000〜5,000円
  中級者（実績5〜20件）  → 1本5,000〜10,000円
  上級者（実績20件以上） → 1本10,000〜20,000円

  月10万円の内訳（中級者の場合）：
  継続A（週2本×5,000円）   = 4万円
  継続B（週1本×8,000円）   = 3.2万円
  スポット案件             = 2.8万円
  合計                    = 10万円
```

単価を上げるポイントは**「ジャンル特化」**。ビジネス系、投資系、教育系など、専門分野に絞ると競合が減って単価が上がる。

「でも専門知識がない」と思った人。AIでリサーチすれば、大抵のジャンルの台本は書けます。ただし事実確認は必ず人間がやること。ここを省くと[AI副業で失敗する人の共通点](/blog/ai-side-job-failures)にハマります。

## YouTube×AI副業の発展形

台本ライターから発展させる道は2つあります。

**発展1：自分のチャンネルを運営する**

台本作成のスキルがあれば、自分のYouTubeチャンネルも運営できる。台本＋AI音声＋AI画像で、完全に一人で動画が作れます。

Instagram Reelsとの相乗効果も高い。[AI×Instagram Reels運用で月5万円稼ぐ方法](/blog/ai-instagram-reels)も参考にしてください。

**発展2：動画制作ディレクター**

台本＋撮影＋編集をまとめてディレクションするポジション。単価は1本3〜5万円に跳ね上がります。
